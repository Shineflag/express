# express
express基本使用,包括注册,登陆,上传,下载,登录权限控制,session持久化,邮件发送,网络爬虫等

线上演示地址 : http://117.40.138.188:15978/before/index

主框架: express 4.16

实时刷新: socket.io 1.4

数据库: mongodb

数据库模块 : mongoose 4.5

上传模块 : multer 1.1 注意:multer1.0版本以后和之前代码差异很大

邮件发送模块 : nodemailer 2.4

网络爬虫 : cheerio 0.20

uploads文件夹 : 上传文件存放

utils文件夹 : 工具类文件

app.config.js文件 : 全局的属性定义
